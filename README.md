Task 3: Exploratory Data Analysis (EDA)
Objective
Perform Exploratory Data Analysis to understand data patterns, feature relationships, outliers, and
important features for prediction.
Dataset
Iris Dataset (Built-in from scikit-learn)
Tools Used
Python, Pandas, Matplotlib, Seaborn
EDA Steps Performed
• Data loading and inspection
• Distribution analysis using histograms
• Categorical analysis using count plots
• Outlier detection using box plots
• Correlation analysis using heatmap
• Feature importance identification
• Visual insights summary
Key Insights
• Dataset contains no missing values
• All classes are equally distributed
• Petal length and petal width show strong correlation
• Setosa species is clearly separable
• Petal features are most important for prediction

Outcome
This task improves understanding of data behavior and prepares the dataset for machine learning
modeling.
